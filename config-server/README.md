# Config Server

Config server that hosts the config files from: [Spring Cloud Config Files](https://github.com/scmathew/spring-cloud-config)

Once running, configs can be found at:
- http://localhost:9000/guestservices/default
- http://localhost:9000/roomservices/default
- http://localhost:9000/reservationservices/default


## Build

### Start Spring Boot App via Maven
```bash
 mvn spring-boot:run
```
